## Installation
1. Go check the [releases](https://github.com/cnrxad/self-hosted-yt-downloader/releases) a look for the [latest](https://github.com/cnrxad/self-hosted-yt-downloader/releases/latest) one.
2. Download the `.exe` file.

## FAQ's
### How does the downloader works?
With the help of the [yt-dlp](github.com/yt-dlp/yt-dlp/) library, a call is made to the YouTube API to download a video, and the downloader formats the file to provide the user with an `.mp4` or `.mp3` file.
